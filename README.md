# MorphSys: a branch-aware contrastive learning framework for neuron morphology graph
This repo holds the implementation of Morphsys described in the paper 
> Wang, Ruoyu, et al. "MorphSys: A branch-aware contrastive learning framework for neuron morphology graphs." Journal of Neural Engineering (2026).
